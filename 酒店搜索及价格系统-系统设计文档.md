### 1.2 核心业务流

```
用户搜索 → 关键词识别 → 命中酒店筛选 → 缓存/实时获取报价 → 加价处理
→ 规则解析(取消/担保/餐食等) → 排序 → 返回前端展示
→ 用户选择 → 预订页实时验价 → 生成订单
```

### 2.3 核心业务流程

#### 2.3.1 列表页搜索报价流程（含三级降级）

```
┌─────────┐     ┌─────────────┐     ┌──────────────┐     ┌───────────────┐
│ 用户搜索  │ ──→ │ 关键词识别     │ ──→ │ 命中酒店筛选   │ ──→ │ 获取报价        │
└─────────┘     └─────────────┘     └──────────────┘     └───────┬───────┘
                                                                │
                                           ┌────────────────────┤
                                           ▼                    ▼
                                  ┌──────────────┐     ┌──────────────┐
                                  │ L1: Redis缓存 │     │ L2: 实时请求   │
                                  │ (分钟~小时级)  │     │ 供应商价格API   │
                                  └──────┬───────┘     └──────┬───────┘
                                         │                    │
                                         │          ┌─────────┤
                                         │          │  API失败/超时
                                         │          ▼
                                         │   ┌──────────────────┐
                                         │   │ L3: DB报价快照     │ ← 降级兜底
                                         │   │ (天级, 标记"参考价") │
                                         │   └──────┬───────────┘
                                         │          │
                                         ▼          ▼
                                    ┌─────────────────────────────────┐
                                    │        加价处理引擎               │
                                    └────────────────┬────────────────┘
                                                     │
                                    ┌────────────────┤
                                    ▼                ▼
                              ┌──────────┐    ┌────────────┐
                              │ 规则解析   │    │ 酒店/房型    │
                              │ (取消/担保 │    │ 排序引擎     │
                              │  /餐食等) │    └──────┬─────┘
                              └─────┬────┘           │
                                    │                │
                                    ▼                ▼
                              ┌─────────────────────────────┐
                              │     组装返回结果 → 前端展示      │
                              └─────────────────────────────┘
```

**三级降级架构说明：**

| 层级 | 数据源 | 数据新鲜度 | 触发条件 | 前端展示 |
|------|--------|-----------|---------|---------|
| **L1** | Redis 缓存 | 分钟~小时级 | 默认首选 | 正常展示价格 |
| **L2** | 实时供应商 API | 实时 | L1 缓存未命中 | 正常展示价格，回写缓存+快照 |
| **L3** | 数据库报价快照 | 天级 | L1 未命中 + L2 请求失败 | 标记"参考价"展示 |
| **L4** | 仅酒店静态信息 | - | L3 快照也过期(>7天) | 只展示酒店信息，不展示价格 |

> **核心原则**：宁可展示一个标注了"仅供参考"的历史价格，也不给用户空白页面。预订环节始终做实时验价（见 2.3.4），确保商业风险可控。

#### 2.3.2 前端5+3秒异步加载机制

```
T=0s  用户点击搜索 → 后端开始请求各供应商API
T=5s  前端倒计时结束 → 展示已获得的报价
T=5-8s 后端继续接收报价
T=8s  前端再做一次刷新 → 展示补充报价
T>8s  后续返回的报价不参与本次展示（但更新缓存）
```

**实现方案：** 使用 WebSocket 或 Server-Sent Events (SSE) 实现增量推送报价。

```java
// 报价推送接口签名
@GetMapping(value = "/api/v1/search/prices/stream", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
public Flux<PriceResultDTO> streamPrices(@Valid SearchRequest request);
```

#### 2.3.3 详情页报价流程

```
用户点击酒店 → 从 detail 缓存获取报价
                ├── L1 缓存命中 → 直接返回
                └── L1 缓存未命中/供应商配置为实时查询
                    → 实时请求价格API (L2)
                       ├── 成功 → 加价处理 → 规则解析 → 返回 (同步回写缓存+快照)
                       └── 失败 → 降级读DB报价快照 (L3)
                           ├── 快照 ≤72h → 返回，前端标记"参考价"
                           ├── 72h<快照≤7天 → 仅展示酒店静态信息 + "查看实时价格"按钮
                           └── 快照 >7天 或无记录 → 仅展示酒店静态信息，不展示价格
```

#### 2.3.4 预订前验价流程

```
用户点击预订 → 实时请求该房型报价
              ├── 报价有效且可订 → 比对价格变动
              │     ├── 变动 ≤ 1元 → 直接进入预订页
              │     └── 变动 > 1元 → 返回变价提示
              └── 报价无效/不可订 → 提示"该房型暂不可订"并刷新缓存
```

#### 2.6.4 报价快照降级兜底方案

当 Redis 缓存(L1)失效 + 供应商 API(L2)不可用时，系统从数据库报价快照(L3)读取历史报价作为降级展示，避免用户看到空页面。

##### 快照写入（异步，零侵入主流程）

```java
@Service
public class PriceSnapshotService {

    @Autowired
    private PriceSnapshotMapper snapshotMapper;

    private static final ExecutorService SNAPSHOT_EXECUTOR = 
        new ThreadPoolExecutor(2, 8, 60L, TimeUnit.SECONDS,
            new LinkedBlockingQueue<>(5000),
            new ThreadPoolExecutor.DiscardOldestPolicy()); // 队列满时丢弃最旧任务

    /**
     * 异步保存报价快照（在CacheUpdateService.updateDetailCache末尾调用）
     * 对主流程零侵入：写入失败仅记日志，不影响正常返回
     */
    public void asyncSaveSnapshot(String supplierId, String hotelId, 
                                   String checkIn, String checkOut,
                                   List<SellRoom> sellRooms, int dataSource) {
        SNAPSHOT_EXECUTOR.submit(() -> {
            try {
                for (SellRoom room : sellRooms) {
                    PriceSnapshot snapshot = PriceSnapshot.builder()
                        .hotelId(hotelId)
                        .roomTypeId(room.getRoomTypeId())
                        .roomTypeName(room.getRoomTypeName())
                        .sellRoomId(room.getSellRoomId())
                        .sellRoomName(room.getSellRoomName())
                        .supplierId(supplierId)
                        .checkInDate(LocalDate.parse(checkIn))
                        .checkOutDate(LocalDate.parse(checkOut))
                        .sellPrice(room.getSellPrice())
                        .costPrice(room.getCostPrice())
                        .currency(room.getCurrency())
                        .breakfastCount(room.getBreakfastCount())
                        .cancelRuleDesc(room.getCancelRuleSummary())
                        .guaranteeDesc(room.getGuaranteeSummary())
                        .paymentType(room.getPaymentType())
                        .promotionTags(JSON.toJSONString(room.getPromotionTags()))
                        .snapshotTime(LocalDateTime.now())
                        .dataSource(dataSource)
                        .build();
                    // UPSERT: 同supplier+hotel+sellRoom+日期仅保留最新一条
                    snapshotMapper.upsert(snapshot);
                }
            } catch (Exception e) {
                log.warn("报价快照写入失败, hotelId={}, supplierId={}", hotelId, supplierId, e);
            }
        });
    }
}
```

##### 快照读取（降级查询）

```java
@Service
public class PriceSnapshotService {

    /**
     * 降级读取DB快照：当L1缓存+L2 API均不可用时调用
     * @param hotelIds   需要降级的酒店ID列表
     * @param checkIn    入住日期
     * @param checkOut   离店日期
     * @return 降级报价Map，key=hotelId，value中 degraded=true
     */
    public Map<String, DegradedPriceInfo> getLatestSnapshot(
            List<String> hotelIds, String checkIn, String checkOut) {
        
        Map<String, DegradedPriceInfo> result = new HashMap<>();
        
        // 查询配置的新鲜度阈值
        int maxAgeHours = snapshotFreshnessConfig.getMaxAgeHours("LIST", "APPROX_REF");
        LocalDateTime threshold = LocalDateTime.now().minusHours(maxAgeHours);
        
        // 批量查询：每个酒店取最新快照中的最低价
        List<PriceSnapshot> snapshots = snapshotMapper.selectLatestByHotels(
            hotelIds, LocalDate.parse(checkIn), LocalDate.parse(checkOut), threshold);
        
        for (PriceSnapshot snap : snapshots) {
            long ageHours = ChronoUnit.HOURS.between(snap.getSnapshotTime(), LocalDateTime.now());
            
            DegradedPriceInfo info = new DegradedPriceInfo();
            info.setHotelId(snap.getHotelId());
            info.setSellPrice(snap.getSellPrice());
            info.setCurrency(snap.getCurrency());
            info.setDegraded(true);                        // 标记为降级数据
            info.setSnapshotTime(snap.getSnapshotTime());  // 快照时间
            info.setSnapshotAgeHours(ageHours);            // 快照年龄(小时)
            
            // 根据快照年龄决定展示模式
            if (ageHours <= 24) {
                info.setDisplayMode("NORMAL_REF");   // "参考价 ¥XXX起"
            } else if (ageHours <= 72) {
                info.setDisplayMode("APPROX_REF");   // "约 ¥XXX起(参考价)"
            } else {
                info.setDisplayMode("NO_PRICE");     // 不展示价格
            }
            
            result.putIfAbsent(snap.getHotelId(), info); // 每个酒店只取最低价
        }
        
        return result;
    }
}
```

```java
        }
```

```java
        Map<String, DegradedPriceInfo> degraded = Collections.emptyMap();
        if (!failedIds.isEmpty()) {

            degraded = priceSnapshotService.getLatestSnapshot(
                failedIds, req.getCheckIn(), req.getCheckOut());
            log.info("降级读取报价快照, 请求{}个酒店, 命中{}个", failedIds.size(), degraded.size());
        }
        
        // 合并三级结果并排序（降级数据排在正常数据之后）
        return mergeAndSort(cached, realtime, degraded);
    }
```

##### 快照数据清理

```sql
-- 定时清理任务(每天凌晨执行)
-- 1. 删除已过入住日期的快照（已无降级展示价值）
DELETE FROM t_price_snapshot WHERE check_in_date < CURDATE();

-- 2. 删除超过7天的快照（即使入住日期未到，数据也已过于陈旧）
DELETE FROM t_price_snapshot WHERE snapshot_time < DATE_SUB(NOW(), INTERVAL 7 DAY);
```

**清理策略说明：**

| 条件 | 动作 | 原因 |
|------|------|------|
| `check_in_date < 今天` | 删除 | 入住日期已过，无展示意义 |
| `snapshot_time > 7天` | 删除 | 数据过于陈旧，展示反而误导用户 |
| 可选：单表超过 500 万行 | 按月分区 | 防止单表过大影响查询性能 |

##### 监控指标

| 指标 | 计算方式 | 报警阈值 |
|------|---------|---------|
| 降级触发率 | L3 触发次数 / 总搜索次数 | > 5% 告警 |
| 降级命中率 | L3 有快照次数 / L3 触发次数 | < 80% 告警（快照覆盖率不足） |
| 快照写入失败率 | 写入异常次数 / 总写入次数 | > 1% 告警 |
| 快照平均年龄 | AVG(NOW() - snapshot_time) | > 48h 关注 |

### 2.8 加价处理引擎

```java
public class MarkupEngine {
    
    /**
     * 加价计算核心逻辑
     * 售价计算: 净房费(roomRate) * (1 + markupRate%) + markupAmount
     * 税费: 原税费 * (1 + markupRate%)  (税费为0时保持为0)
     * 立减: 原立减 * (1 + markupRate%)
     * 附加费: 不做加价处理
     * 国内酒店四舍五入取整, 国际保留两位小数
     */
    public SellingPrice calculate(SupplierPrice supplierPrice, 
                                   PriceStrategy strategy, 
                                   boolean isDomestic) {
        BigDecimal rate = strategy.getMarkupRate().divide(BigDecimal.valueOf(100));
        BigDecimal amount = strategy.getMarkupAmount();
        
        // 净房费
        BigDecimal newRoomRate = supplierPrice.getRoomRate()
            .multiply(BigDecimal.ONE.add(rate))
            .add(amount);
        
        // 税费
        BigDecimal newTaxAndFee = supplierPrice.getTaxAndFee()
            .multiply(BigDecimal.ONE.add(rate));
        
        // 立减
        BigDecimal newDiscount = supplierPrice.getDiscountAmount()
            .multiply(BigDecimal.ONE.add(rate));
        
        // 售价 = 净房费 + 税费 - 立减
        BigDecimal sellingPrice = newRoomRate.add(newTaxAndFee).subtract(newDiscount);
        
        // 销售总价 = 售价 + 原附加费
        BigDecimal totalPrice = sellingPrice.add(supplierPrice.getSurcharge());
        
        // 精度处理
        if (isDomestic) {
            sellingPrice = sellingPrice.setScale(0, RoundingMode.HALF_UP);
            totalPrice = totalPrice.setScale(0, RoundingMode.HALF_UP);
        } else {
            sellingPrice = sellingPrice.setScale(2, RoundingMode.HALF_UP);
            totalPrice = totalPrice.setScale(2, RoundingMode.HALF_UP);
        }
        
        return new SellingPrice(newRoomRate, newTaxAndFee, newDiscount, 
                                sellingPrice, totalPrice);
    }
    
    /**
     * 取消规则罚金重算(加价后)
     * 售价 >= 原房费+原税费-原立减 时的计算方式
     * 售价 < 原房费+原税费-原立减 时的特殊处理
     */

                                          SupplierPrice original) { ... }
    
    /**
     * 价格策略匹配
     * 优先级: 特殊策略(命中多条时取价低者) > 全局策略
     */
    public PriceStrategy matchStrategy(Long supplierId, String channel) {
```

```java
        int starScore = calculateStarScore(hotel);         // 20/30/40/50
        int recommendScore = calculateRecommendScore(hotel, destinationCode); // n*1000
        
        return imageScore + ratingScore + reviewScore + starScore + recommendScore;
    }
    
    /**
     * 全局排序规则
     * 有报价可订 > 有报价已订完 > 无报价
     * 子规则按搜索条件分类处理
     */
    public List<Hotel> sort(List<Hotel> hotels, SearchRequest request) {
        // 1. 按可订状态分组
        Map<HotelStatus, List<Hotel>> grouped = groupByStatus(hotels);
        
        // 2. 各组内子排序
        List<Hotel> available = subSort(grouped.get(AVAILABLE), request);
        List<Hotel> soldOut = subSort(grouped.get(SOLD_OUT), request);
        List<Hotel> noPrice = subSort(grouped.get(NO_PRICE), request);
        
        // 3. 前20名档次比例调整 (2:4:3:1)
        available = adjustTop20StarRatio(available);
        
        // 4. 合并
        List<Hotel> result = new ArrayList<>();
        result.addAll(available);
        result.addAll(soldOut);
        result.addAll(noPrice);
        return result;
    }
    
    /**
     * 子排序逻辑(按搜索条件分类)
     */
    private List<Hotel> subSort(List<Hotel> hotels, SearchRequest request) {
        KeywordType kwType = request.getKeywordType();
        
        switch (kwType) {
            case NONE:          // 无关键字: 按Rank降序

            case GEO:           // 地理位置: 按距离分区间,区间内按Rank

                return sortByRank(hotels);
            case GEO_FILTER:    // 地理+筛选: 匹配度>Rank
                return sortByMatchDegThenRank(hotels, request);
            default:
                return sortByRank(hotels);
```

```java
public class CancelRuleEngine {
    
    /**
     * 取消规则解析
     * 1. 时区转换(供应商时区/城市时区→北京时间)
     * 2. before字段解析
     * 3. 过期规则过滤
     * 4. 话术生成
     */
    public CancelPolicy parse(List<CancelRule> rules, BookingContext context) {
        // 1. 时区转换
        rules = convertTimezone(rules, context);
        
        // 2. before字段解析 (距入住日24:00前X小时)

        // 3. 解析后时间往前推4小时(实际执行时间)
        rules = adjustExecutionTime(rules);
        
        // 4. "DD日00:00"提前1分钟显示为"DD-1日23:59"
        rules = adjustMidnightDisplay(rules);
        
        // 5. 过滤已过期规则
        rules = filterExpiredRules(rules, LocalDateTime.now());
        
        // 6. 生成话术
        return generateCancelPolicyText(rules, context);
    }
    
    /**
     * 违约金计算
     * 支持: 不扣款/按金额/按百分比/扣X晚
     * 多间房时的总罚金计算
     */
    public BigDecimal calculatePenalty(CancelRule rule, PriceInfo priceInfo, 
                                        int roomCount) {
        switch (rule.getPenaltyType()) {
            case NO_PENALTY:
                return BigDecimal.ZERO;
            case BY_AMOUNT:
                return rule.getPenaltyValue().multiply(BigDecimal.valueOf(roomCount));
            case BY_PERCENTAGE:
                BigDecimal baseAmount = calculateBaseForPenalty(priceInfo, roomCount);
                return baseAmount.multiply(rule.getPenaltyValue().divide(HUNDRED));
            case BY_NIGHTS:
                return calculateNightsPenalty(rule, priceInfo, roomCount);
            default:
                return BigDecimal.ZERO;
        }
    }
}
```

```java
/**
 * 供应商自动上下线定时任务
 * 每分钟执行一次
 */
@XxlJob("supplierAutoOnOffHandler")
public void supplierAutoOnOff() {
    LocalDateTime now = LocalDateTime.now();
    DayOfWeek today = now.getDayOfWeek();
    LocalTime currentTime = now.toLocalTime();
    
    List<Supplier> suppliers = supplierService.getAllNon24x7Suppliers();
    
    for (Supplier supplier : suppliers) {
        WorkSchedule schedule = getScheduleForDay(supplier.getId(), today);
        if (schedule == null) {
            // 当天无工作时间，下线
            supplierService.autoOffline(supplier.getId());
            continue;
        }
        
        LocalTime endTime = schedule.getEndTime();
        LocalTime startTime = schedule.getStartTime();
        
        // 不是 23:59 结束且次日 00:00 开始的情况
        // 在结束前15分钟提前下线
        if (shouldAutoOffline(currentTime, endTime, startTime)) {
            supplierService.autoOffline(supplier.getId());
        }
        
        // 到上班时间恢复上线
        if (shouldAutoOnline(currentTime, startTime)) {
            supplierService.autoOnline(supplier.getId());
        }
    }
}
```

```javascript
// stores/searchStore.js
import { defineStore } from 'pinia'

export const useSearchStore = defineStore('search', {
    state: () => ({
        // 目的地
        destination: {
            code: '',
            name: '',           // 如 "上海"
            nameEn: '',
            type: 'CITY',       // CITY/DISTRICT/CITY_TAG/POI
            isCurrentLocation: false,
            latitude: null,
            longitude: null
        },
        // 日期
        checkInDate: null,       // dayjs 对象
        checkOutDate: null,
        nightCount: 1,
        // 入住信息
        roomCount: 1,
        adultCount: 2,
        childCount: 0,
        childAges: [],
        // 关键词
        keyword: '',
        keywordType: '',
        keywordRefId: '',
        // 档次价格
        starLevels: [],
        priceRange: { min: null, max: null },
        // 历史
        searchHistory: [],       // 最近8条
        destHistory: [],         // 最近8个目的地
    }),

    getters: {
        isDomestic: (state) => state.destination.type !== 'INTERNATIONAL',
        // 格式化为截图中的 "04-05" 样式
        checkInDisplay: (state) => state.checkInDate?.format('MM-DD'),
        checkOutDisplay: (state) => state.checkOutDate?.format('MM-DD'),
    },

    actions: {
        setDestination(dest) {
            this.destination = dest
            this.keyword = ''
            this.starLevels = []
            this.priceRange = { min: null, max: null }
            this.addDestHistory(dest)
        },
        setDates(checkIn, checkOut) {
            this.checkInDate = checkIn
            this.checkOutDate = checkOut
            this.nightCount = checkOut.diff(checkIn, 'day')
        },
        // 从历史恢复搜索条件（首页交互逻辑）
        restoreFromHistory() {
            const history = getFromStorage('lastSearch')
            if (!history) return
            if (dayjs(history.checkInDate).isBefore(dayjs(), 'day')) {
                const d = this.isDomestic
                    ? { ci: dayjs().add(1,'day'), co: dayjs().add(2,'day') }
                    : { ci: dayjs().add(15,'day'), co: dayjs().add(16,'day') }
                this.checkInDate = d.ci
                this.checkOutDate = d.co
            } else {
                this.checkInDate = dayjs(history.checkInDate)
                this.checkOutDate = dayjs(history.checkOutDate)
            }
            this.destination = history.destination
            this.keyword = history.keyword || ''
        },
        addDestHistory(dest) {
            this.destHistory = [dest, ...this.destHistory
                .filter(d => d.code !== dest.code)].slice(0, 8)
        }
    }
})
```

#### 3.6.0 降级态 UI 展示规范

当报价数据来源于 L3 数据库快照（降级兜底）时，前端必须明确标识数据为"参考价"，避免用户误解。后端通过接口字段 `degraded=true` + `displayMode` 传递降级状态。

**降级态价格展示规则：**

| displayMode | 快照年龄 | 列表页展示 | 详情页展示 |
|-------------|---------|-----------|-----------|
| `NORMAL_REF` | ≤24h | `参考价 ¥XXX起` + 浅灰标签 | 房型卡片正常展示，价格旁加"参考价"标签 |
| `APPROX_REF` | 24h~72h | `约 ¥XXX起` + "参考价"标签 | 房型卡片展示，价格显示"约 ¥XXX"，顶部提示条 |
| `NO_PRICE` | >72h | 仅展示酒店名/星级/图片，不展示价格 | 仅展示酒店静态信息 + "查看实时价格"按钮 |

```javascript
import axios from 'axios'
import { showToast } from 'vant'

const service = axios.create({
    baseURL: import.meta.env.VITE_API_BASE_URL,
    timeout: 30000,
    headers: { 'Content-Type': 'application/json' }

service.interceptors.request.use(config => {
    const token = localStorage.getItem('token')
    if (token) config.headers.Authorization = `Bearer ${token}`
    config.headers['X-Channel'] = 'H5'
    config.headers['X-Session-Id'] = getSessionId()
    return config
})

service.interceptors.response.use(
    response => {
        const { code, data, message } = response.data
        if (code === 200) return response.data
        showToast(message || '请求失败')
        return Promise.reject(new Error(message))
    },
    error => {
        if (error.code === 'ECONNABORTED') {
            showToast('网络请求超时，请重试')
        } else if (!navigator.onLine) {
            showToast('网络不可用，请检查网络连接')
        } else {
            showToast('服务器异常，请稍后重试')
        }
        return Promise.reject(error)
    }
)

export default service
```

### 3.8 数据埋点设计

```javascript
// utils/tracker.js
class Tracker {
    constructor() { this.sessionId = this.getOrCreateSessionId() }

    trackPageView(pageType) {
        this.send('page_view', { pageType, sessionId: this.sessionId })
    }
    trackSearch(searchParams) {
        this.send('search', { ...searchParams, sessionId: this.sessionId })
    }
    trackHotelClick(hotelData) {
        this.send('hotel_click', { ...hotelData, sessionId: this.sessionId })
    }
    trackBookingClick(bookingData) {
        this.send('booking_click', { ...bookingData, sessionId: this.sessionId })
    }
    trackFilterClick(filterType, value) {
        this.send('filter_click', { filterType, value, sessionId: this.sessionId })
    }

    send(event, data) {
        const payload = { event, data, timestamp: Date.now(), url: location.href }
        if (navigator.sendBeacon) {
            navigator.sendBeacon('/api/v1/track', JSON.stringify(payload))
        } else {
            fetch('/api/v1/track', { method:'POST', body:JSON.stringify(payload), keepalive:true })
        }
    }
    getOrCreateSessionId() {
        let sid = sessionStorage.getItem('session_id')
        if (!sid) { sid = crypto.randomUUID(); sessionStorage.setItem('session_id', sid) }
        return sid
    }
}
export const tracker = new Tracker()
```

### B. 入住人分配算法

```javascript
// utils/guest-allocator.js
/**
 * 入住人分配到各房间
 * @param {number} roomCount - 房间数
 * @param {number} adultCount - 成人数
 * @param {number} childCount - 儿童数
 * @param {number[]} childAges - 儿童年龄(降序)
 * @returns {Array<{adults: number, children: number[]}>}
 */
export function allocateGuests(roomCount, adultCount, childCount, childAges) {
    const rooms = Array.from({ length: roomCount }, () => ({ adults: 0, children: [] }))
    
    // 1. 分配成人: 每间房 floor(adultCount/roomCount) 人
    const baseAdults = Math.floor(adultCount / roomCount)
    const remainAdults = adultCount % roomCount
    
    for (let i = 0; i < roomCount; i++) {
        rooms[i].adults = baseAdults
    }
    // 剩余成人按房间号1~n顺序分配
    for (let i = 0; i < remainAdults; i++) {
        rooms[i].adults++
    }
    
    // 2. 分配儿童: 按年龄从大到小排序
    const sortedAges = [...childAges].sort((a, b) => b - a)
    
    // 找到最后一个多分配到成人的房间索引y
    let startRoom = remainAdults > 0 ? remainAdults : 0
    
    // 第一轮正向分配: 从startRoom开始
    let childIdx = 0
    for (let i = startRoom; i < roomCount && childIdx < sortedAges.length; i++) {
        rooms[i].children.push(sortedAges[childIdx++])
    }
    // 反向分配(从最后一间到第一间)
    for (let i = roomCount - 1; i >= 0 && childIdx < sortedAges.length; i--) {
        rooms[i].children.push(sortedAges[childIdx++])
    }
    // 继续循环正向分配直到分完
    while (childIdx < sortedAges.length) {
        for (let i = 0; i < roomCount && childIdx < sortedAges.length; i++) {
            rooms[i].children.push(sortedAges[childIdx++])
        }
    }
    
    return rooms
}
```
